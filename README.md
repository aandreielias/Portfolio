# Portfolio Project

This repository contains the source code for the portfolio application.

## Structure

- **backend/**: Java Spring Boot application (API & Legacy).
- **frontend/**: Svelte + Vite web application.

## Getting Started

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. `./mvnw spring-boot:run`
